# 像素海鸥地理大冒险 - RPG版

## 游戏简介
这是一款面向七年级学生的像素风格RPG地理学习游戏，通过探索多样化的地图场景和互动地形元素，帮助学生以趣味化的方式学习和巩固地理知识。

## 核心功能

### RPG风格地图系统
- **神秘森林**: 绿色森林场景，树木障碍物，学习森林地理知识
- **水晶洞穴**: 灰色洞穴场景，水晶障碍物，学习地质地貌知识  
- **火山遗迹**: 棕色火山场景，火山岩障碍物，学习火山地质知识

### 地形元素系统
- **树木**: 森林地图的主要障碍，答对后变绿色
- **水晶**: 洞穴地图的发光障碍，答对后变蓝色
- **火山岩**: 火山地图的岩石障碍，答对后变棕色
- **装饰元素**: 增加地图丰富度的小型地形
- **路径系统**: 引导玩家从起点到终点的明确路线

### 因地制宜的问题系统
- **森林地图**: 关于森林分布、树木类型、生态系统的地理问题
- **洞穴地图**: 关于水晶成分、钟乳石形成、喀斯特地貌的地质问题
- **火山地图**: 关于火山岩类型、火山带分布、熔岩成分的火山地质问题

### 游戏机制
- **起点终点系统**: 每张地图有明确的起点(S)和终点(E)标记
- **路线引导**: 虚线箭头指示前进方向
- **碰撞检测**: 碰到障碍物触发相关问题
- **进度追踪**: 实时显示当前地图完成进度
- **地图解锁**: 完成当前地图所有知识点后自动解锁下一张

## 操作说明
- **方向键**: 使用 ↑↓←→ 控制海鸥在地图上移动
- **鼠标点击**: 选择答案选项
- **重新开始**: 游戏结束后点击重新开始按钮
- **地图滚动**: 大地图支持鼠标滚轮和拖拽滚动

## 地理知识点
游戏包含15个精心设计的因地制宜问题：
- **森林地理** (5题): 森林分布、树木类型、生态系统
- **洞穴地质** (5题): 水晶成分、钟乳石、喀斯特地貌
- **火山地质** (5题): 火山岩类型、火山带、熔岩成分

## 技术特点
- **RPG地图设计**: 1200x600大地图，支持滚动探索
- **多样化地形**: 树木、水晶、火山岩等像素风格地形
- **视觉反馈**: 已完成障碍物变色，实时进度显示
- **路径引导**: 清晰的起点终点和方向指示
- **响应式布局**: 适配不同屏幕尺寸

## 使用方法
1. 直接在浏览器中打开 `index.html` 文件
2. 使用方向键控制海鸥从起点(S)移动
3. 沿着路径探索地图，碰到障碍物时回答地理问题
4. 答对消除障碍物（变色），答错扣血
5. 完成所有障碍物后到达终点(E)解锁下一张地图
6. 完成所有地图或生命值耗尽时游戏结束

## 教育价值
- **情境化学习**: 不同地图场景对应不同地理知识点
- **视觉化教学**: 地形元素与地理概念直接关联
- **渐进式学习**: 从森林到洞穴到火山，难度递增
- **即时反馈**: 错误立即纠正，正确立即奖励
- **探索乐趣**: RPG式地图探索增加学习动力
